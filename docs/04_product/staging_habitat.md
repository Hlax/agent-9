# Staging Habitat

The Staging Habitat is a sandbox environment where artifacts
and presentation concepts can be previewed before public release.

---

# 1. Purpose

The staging habitat allows testing of:

- layouts
- presentation styles
- collections
- narrative sequences
- interactive ideas

Before publishing publicly.

---

# 2. What Appears in Staging

Artifacts approved for staging may include:

- concept artifacts
- visual outputs
- writing
- experimental layouts

Artifacts do not need to be final.

---

# 3. Use Cases

Examples:

- preview gallery layout
- test narrative sequence
- prototype interactive experience

---

# 4. Approval

Artifacts must be approved for staging.

Staging does not equal public release.

---

# 5. Relationship to Studio

Studio controls:

- staging selection
- artifact approval
- layout configuration

---

# 6. Relationship to Public Habitat

Public habitat only shows curated outputs.

Staging is a preview environment.
