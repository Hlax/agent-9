# Release Archive

The Release Archive preserves artifacts that have been published
to the public habitat.

It functions as the Twin's public history.

---

# 1. Purpose

The release archive allows:

- historical browsing
- version tracking
- curated collections

---

# 2. Published Artifacts

Artifacts appear in the release archive once:

approved_for_publication  
and  
published

---

# 3. Archive Organization

Possible structures:

chronological  
project-based  
theme-based

---

# 4. Version Preservation

Published artifacts should remain historically stable.

If an artifact changes significantly,
a new version should be created rather than replacing the original.

---

# 5. Archive Value

The archive allows the Twin's evolution to remain visible across time.
